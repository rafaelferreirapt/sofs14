Detailed Description

The SOFS14 block/cluster display tool.

(interface file)

It displays preselected blocks/clusters of the storage device, supposed to contain file system data or metadata. With it, the storage device is envisaged as an array of blocks, some of them joined in clusters.

The following display formats are available:

  • block/cluster contents as hexadecimal data
  • block/cluster contents as ascii/hexadecimal data
  • block/cluster contents both as hexadecimal and ascii data
  • block contents as superblock data
  • block contents as a sub-array of inode entries
  • cluster contents as a byte stream
  • cluster contents as a sub-array of directory entries
  • cluster contents as a sub-array of data cluster references.

SINOPSIS: 

                   showblock_sofs14 OPTIONS supp-file
              OPTIONS:
               -x blockNumber   --- show the block contents as hexadecimal data
               -X clusterNumber --- show the cluster contents as hexadecimal data
               -b blockNumber   --- show the block contents as ascii/hexadecimal data
               -B clusterNumber --- show the cluster contents as ascii/hexadecimal data
               -a blockNumber   --- show the block contents both as hexadecimal and ascii data
               -A clusterNumber --- show the cluster contents both as hexadecimal and ascii data
               -s blockNumber   --- show the block contents as superblock data
               -i blockNumber   --- show the block contents as a sub-array of inode entries
               -T clusterNumber --- show the cluster contents as a byte stream
               -D clusterNumber --- show the cluster contents as a sub-array of directory entries
               -R clusterNumber --- show the cluster contents as a sub-array of data cluster references
               -h               --- print this help.
Remarks
All cluster and block numbers in OPTIONS are physical numbers (indexes of the array of blocks that represent the storage device at this abstraction level). Furthermore, a cluster number is the number of the first block that comprises it.
